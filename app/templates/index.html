Regex Injection Endpoint - /user_id/$UID

  • The intended functionality of this endpoint is to look up information about users based on user ID. Navigate to /user_id/$someUID to return information about that user.
  • This is a regex injection, so the payload is very simple. If you navigate to user_id/.*, it should return a list of user IDs for all the users in the database.
